# 🎮 操作界面规划：GTA5 × 《我的世界》融合

> 目标：在现有"数据城市"上套一层 **GTA5 式开放世界操作壳** + **《我的世界》式建造/物品系统**，均衡融合、两层无缝切换。尽量复用已有系统。

## 一、两层操作（已有雏形）
现有「🛰 上帝视角 ⇄ 👣 进入地面」正好对应：
- **🛰 指挥层（俯视）** —— 偏《我的世界》创造 + 经营：快捷栏在地图上放置/拆除建筑、管财政、定目标。
- **🎮 化身层（地面，第一/第三人称）** —— 偏 GTA5：操控一个市民跑动、上车、互动、做任务。
- `B` 键在两层切换（即现有"进入地面/上帝视角"）。

## 二、GTA5 → 操作壳
| 系统 | 做法 | 复用现有 |
|---|---|---|

| ❤️ 角色 HUD | 雷达旁环形 生命/体力/饥饿 + 金钱 + 城市警戒条 | 接市民需求 mood/energy/fullness、money、危机系统 |

## 四、统一键位（建议）
`WASD` 移动 · `鼠标` 视角 · `1–9` 选工具 · `滚轮` 切工具(地面)/缩放(俯视) · `左键` 放置/互动 · `右键` 拆除 · `E` 进出建筑/上下车 · `按住 Q` 动作轮盘 · `F5` 第一/三人称 · `M` 大地图 · `Tab` 切换角色 · `B` 指挥/化身切换 · `Shift` 奔跑 · `空格` 跳

## 五、HUD 布局
- 左下：雷达 + 环形 生命/体力/饥饿 + 金钱
- 底部中：快捷栏 9 格 + 当前工具名
- 正中：准星 + 目标格高亮
- 左上：通知 / 新闻 feed
- 右上：年份/时间 + 当前任务
- 角落：动作轮盘提示 · 创造/生存模式标

## 六、落地阶段（按性价比）
1. ✅ **界面骨架**：雷达小地图 + 生命/体力/饥饿+金钱 + 快捷栏 1-8 + 准星 + 创造/生存模式标 + F5 视角 + B 切换两层。**已实现**
2. ✅ **建造交互**：化身层准星指地→半透明幽灵预览，左键放置 / 右键(或拆除工具)拆除，吸附网格。**已实现**
3. ✅ **GTA 动作**：`F` 上车驾驶（专属座驾，下车恢复步行）+ 按住 `Q` 动作轮盘（上帝视角/上下车/切换角色/视角/地图/创造生存）+ `Tab` 就近切换附身角色。**已实现**
4. ✅ **地图/任务**：`M` 大地图（建筑/绿地/市民/玩家点）+ 点击设路点 → 雷达光点 + 世界中旋转**信标光柱** + 到达提示。**已实现**
5. ✅ **打磨**：创造/生存模式（生存扣市库、创造免费）+ 科技解锁（风电/电厂需"近未来"纪元，快捷栏显示 🔒）。**已实现**

> 全部 5 阶段已落地。键位：`WASD` 移动·`鼠标` 视角·`1-8`/滚轮 选工具·`左键` 放置·`右键` 拆除·`E` 进出建筑·`F` 上下车·`Tab` 切人·`按住Q` 轮盘·`F5` 视角·`M` 地图·`B` 两层切换。

> 复用比例高：雷达=改造现有小地图；HUD=接现有需求/市库；快捷栏=封装现有 buildAction；任务=接现有目标系统；驾驶=接现有车流。新增主要是：放置预览、动作轮盘、上车、大地图、视角切换。
