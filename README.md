## 🌆 city.html —— 数据城市 · 未来演化模拟器

以**你所在城市的真实数据**为起点，建起一座拥有完整基建的城市，看它自动演化向未来，你可以随时干预、亲手建设。

### 接入的真实数据（全部免费、无需密钥）

- 📍 **真实定位 + 时间**（被拒则用默认"北京"，左下角可输入城市名切换）——决定天色、季节、昼夜
- 🌤 **天气 + 空气质量**（Open-Meteo）——真实晴雨雪雾；空气质量决定雾霾浓度并初始化污染
- 🌍 **全球地震**（USGS）——真实地震触发城市灾害：镜头震动、建筑受损、经济与幸福下降，之后重建
- 💱 **汇率**（open.er-api.com）+ **加密货币**（CoinGecko）——驱动经济指数

### 🗺 真实地图建城（OpenStreetMap）

进入时会用**你所在位置**（定位被拒则用默认城市，可在左下角输入城市名切换）从 [OpenStreetMap/Overpass](https://wiki.openstreetmap.org/wiki/Overpass_API) 拉取**真实数据**，把城市建成真实样子：

- 🏢 **真实建筑轮廓**：按真实 footprint + 高度挤出成 3D（约 200+ 栋）
- 🛣 **真实路网**：道路按等级生成不同宽度
- 🏷 **真实地名**：餐厅/商店/学校/银行等真实地点带名字浮标（如「中国银行」「王府井喜悦购物中心」），并作为市民生活的场所
- 🌳 真实公园绿地
- 🚗 **真实路网交通**：由真实道路构建路网图（A* 寻路），市民**沿真实街道行走**，并有车辆在路网上行驶、车距过近会**减速形成拥堵**并反哺交通指标
- ✨ **画质**：bloom 辉光后期处理（夜晚霓虹/路灯/太阳更有光感）

免密钥、浏览器直接获取；拉取失败或数据不足时自动回退到下面的程序生成城市。

### 城市基建（程序生成模式 / 真实数据不足时）

- 🏙 **功能分区**：市中心商务区（玻璃高楼）、住宅区、工业区（工厂带烟囱、冒烟）
- 🏥 医院 · 🎓 学校 · 🏟 体育场 · ⚡ 发电厂 · 📡 地标通信塔（顶端红灯闪烁）
- 🌉 河流 + 跨河桥梁　🛣 高架环路（车辆穿行）　🚄 铁路 + 行驶的列车 + 站房　✈️ 机场 + 绕城飞机
- 🍃 风力发电机（转动）· ☀️ 光伏电站　💡 路灯（夜晚亮起）· 🌳 公园绿地

### 五大模拟系统（接入演化）

| 系统 | 受什么影响 | 影响什么 |
| --- | --- | --- |
| ⚡ 电力 | 发电厂 / 风电 / 光伏 vs 人口与建筑 | 供电不足→经济、人口受损 |
| 💼 就业 | 商务 + 工业岗位 vs 人口 | 经济、幸福 |
| 🏥 医疗 | 医院数量 vs 人口、空气 | 人口增长、幸福 |
| 🎓 教育 | 学校数量 vs 人口 | 加速科技 |
| 🚦 交通 | 人口 vs 道路 / 高架 / 铁路 | 拥堵→幸福下降 |

> 关键玩法：城市越大，越要持续扩建医院/学校/电厂，否则服务跟不上、幸福度会掉。

### 自动演化

年份推进（约 3 秒/年 @1×），人口、经济、科技、污染、幸福度随之变化；城市从「当代」混凝土逐步生长为「近未来 → 未来都市 → 超未来」霓虹都市，建筑实时长高、换风格。

### 市民与操控

- 🧍 **会自己生活的市民 NPC**（数十位）：每人有姓名、年龄、职业、家与工作地点，以及精力/饱腹/社交/娱乐/金钱五项**需求**和**个性权重**。他们用**效用制 AI + Smart Object** 决策（借鉴《模拟人生》）——场所"广播"自己能满足哪些需求，市民按需求紧迫度（接近耗尽时陡升的响应曲线）+ 时段 + 距离 + 个性打分，并在最优的几个动作里**加权随机**，因此行为有个体差异、不机械。点选某人可看到他**当前在做什么、为了满足哪项需求**。
- 👣 **进入地面**：点底部「进入地面」在「上帝视角看全城」与「地面第三人称」之间切换。地面模式 `WASD` 移动、`Shift` 奔跑、拖动鼠标转视角。
- 🖱 **点选市民干预**：点击任意市民查看其人生与需求，可**附身操控**、给钱、换工作、鼓励，或一键派TA去 回家/上班/餐厅/公园——改变他/她的生活轨迹。
- 🚪 **进入建筑**：地面模式走到住宅/餐厅/商店/公司门口，按 **E** 即可进入简单室内（家具、室内 NPC），再按 E 离开。
- 💬 **社交与记忆**：市民相遇会**交谈**（冒出对话气泡）、**结识彼此并记住**（认识 N 人、最近和谁聊过），社交满足度上升；点选可查看他的社交记录。

### 进阶系统（最新）

- 🗺️ **数据图层**：点底部「数据图」切换 关闭 / 污染热力 / 地价——基于真实工业、绿地、交通做**污染元胞扩散**与**地价**计算，半透明叠加在城市上（数据地球既视感）。
- 🧠 **生成式智能体-lite**：市民有**记忆流**（记住做过的事、和谁聊过）、会**反思**（如"想去看看新地方""得多挣点钱了"），交谈时冒出**按状态/关系生成的中文对话**。`npcLine()` 是单一接口，可替换为真实 LLM（WebLLM 本地或后端代理）。
- 🚇 **真实公交/轨道线**：从 OSM 取你所在地的真实地铁/轻轨/铁路线绘制成彩色线路，并有列车往返运行。
- ✨ **画质**：bloom 辉光 + ACES 电影色调 + 环境反射(PBR) + SSAO 环境光遮蔽。
- 🚶 **熙攘人群**：在 80 个"完整市民"之外，额外叠加一层**实例化环境人群**（约 250–700 路人，沿真实路网行走，一次 draw call），城市熙熙攘攘。
- 🤖 **本地 AI 对话（可选）**：点底部「AI对话」可启用浏览器内 **WebLLM**（需 WebGPU，首次下载小模型约数百 MB）；启用后点选市民→「交谈」由**本地大模型**生成符合其身份/心情的台词。未启用或不支持时自动回退模板对话（`npcLine()`）。不上传任何数据、无需密钥。
- 🌓 **自适应高清阴影**：4096 阴影贴图，阴影范围随镜头远近自动收紧——你看哪儿、哪儿的阴影就更锐利。
- 👥 **真实人口（普查级）**：用反向地理编码 + Open-Meteo 地理编码取你所在城市的**真实人口**（如北京约 1896 万），显示在左上数据栏，并作为模拟的起始人口（服务容量按比例缩放，保持平衡）。

### 玩法 · 经营 · 人生（最新一批）

- 🎯 **目标与危机**：选剧本目标（宜居之城 / 碳中和 / 大都会 / 繁荣经济）——实时进度条 + 倒计时 + 达成计分 / 超时失败；随机危机事件（热浪 / 衰退 / 电荒 / 公共卫生）考验你的应对。
- 💰 **市库经营**：城市有财政收支（税收 vs 维护），**建造消耗市库**、赤字会拖累幸福度——真正的经营压力。
- 👴 **生老病死 · 人生愿望**：市民会**变老、离世，由新生儿/迁入者补位**（整代更替）；每人有人生愿望（买房 / 升职 / 找对象 / 创业…），达成会广播；这些汇成左下角的**城市新闻流**。
- 🔊 **城市音效**：程序化 WebAudio——车流 / 人群随人口昼夜起伏、雨声、地震轰鸣 + 警报、轻氛围和弦；可一键静音。
- 💾 **存档 / 分享**：保存城市到本地、读取重建；📷 一键导出**城市快照 PNG**（"城市明信片"）。
- ⚙️ **画质档位**：高 / 中 / 低（像素比 + 人群密度），照顾移动端与弱机。
- 🌊 **真实水体**：从 OSM 取真实湖泊 / 河面，渲染成蓝色水面。

> 真正需要后端/大重构的项（GPGPU 数万人群、Web Worker、SRTM 地形高程、实时新闻流、真·CSM）暂未做，已在 ROADMAP 标注。
